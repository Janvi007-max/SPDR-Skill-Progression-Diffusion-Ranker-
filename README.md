## 📌 Overview

SPDR (Skill Progression Diffusion Ranker) is an AI-powered candidate ranking system built for the Redrob AI Hiring Challenge. Instead of ranking candidates by keyword overlap, SPDR predicts their **future role fit** using a role-conditioned skill progression simulation, helping recruiters identify high-potential talent rather than just the best-looking resumes.

---

## ✨ Key Features

* Predicts **future candidate fit** instead of current resume similarity.
* Detects keyword stuffing through **Title ↔ Career Corroboration**.
* Filters adversarial and inconsistent profiles.
* Explainable AI with transparent ranking reasons.
* Interactive recruiter dashboard with visual insights.
* Lightweight CPU-only pipeline.
